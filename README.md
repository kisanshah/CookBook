# CookBook
### An application to share recipe.
### Components used :-

* MVVM Architecture
* Dependency Injection (Hilt)
* RecyclerView
* Firebase Authentication
* Firebase Database
* Material Design
* Glide
* Jetpack Navigation with SafeArgs
